---

### Comparison Overview

| Feature | Responses API | Chat Completions API |
| :--- | :--- | :--- |
| **Positioning** | Modern proxy-focused API | Industry standard |
| **Built-in Tools** | Web Search, File Search, etc. | Custom functions only |
| **Ease of Use** | Higher (automated tool handling) | Standard |
| **Use Case** | Action-oriented applications | General chat & legacy apps |

### When to Use the Responses API

The Responses API is recommended for new projects that require integrated capabilities like **Web Search** or complex **Reasoning Effort** controls without managing manual tool loops.
