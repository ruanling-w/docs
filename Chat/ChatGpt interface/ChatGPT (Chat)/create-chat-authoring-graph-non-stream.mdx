---
title: "Create Chat Image Authoring (Non-Streaming)"
api: "POST /v1/chat/completions"
description: "Generates or enhances images within a chat context using specialized models."
---

Specialized models like `gpt-4o-image-vip` allow you to enhance or create images based on conversation text and external image URLs.

### Request Body

<ParamField body="model" type="string" required>
  The model ID (e.g., `gpt-4o-image-vip`).
</ParamField>

<ParamField body="messages" type="array" required>
  List of messages, including text prompts and image references.
</ParamField>
