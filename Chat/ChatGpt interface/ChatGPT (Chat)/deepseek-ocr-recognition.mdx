---
title: "DeepSeek OCR Recognition"
api: "POST /v1/chat/completions"
description: "Extract text from images using DeepSeek's optimized OCR model."
---

DeepSeek OCR is a specialized model for high-accuracy text extraction from images.

### Request Body

<ParamField body="model" type="string" required>
  Must be set to `deepseek-ocr`.
</ParamField>

<ParamField body="messages" type="array" required>
  The conversation history. Note the specific system prompt required for OCR:
  
  ```json
  [
    {
      "role": "system",
      "content": "<image>\nFree OCR."
    },
    {
      "role": "user",
      "content": [
        {
          "type": "image_url",
          "image_url": {
            "url": "https://example.com/image.jpg"
          }
        }
      ]
    }
  ]
  ```
</ParamField>

<ParamField body="stream" type="boolean" default="false">
  Whether to stream the OCR results.
</ParamField>
