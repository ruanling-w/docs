Structured Outputs ensure the model consistently generates responses that match your provided JSON Schema.

### Request Body

<ParamField body="model" type="string" required>
  The model ID (e.g., `gpt-4o`).
</ParamField>

<ParamField body="messages" type="array" required>
  Conversation history.
</ParamField>

<ParamField body="response_format" type="object" required>
  The format of the response.
  <Expandable title="properties">
    <ParamField body="type" type="string" required>
      Set to `json_schema`.
    </ParamField>
    <ParamField body="json_schema" type="object" required>
      <Expandable title="schema properties">
        <ParamField body="name" type="string" required>
          The name of the schema.
        </ParamField>
        <ParamField body="schema" type="object" required>
          The actual JSON Schema definition.
        </ParamField>
        <ParamField body="strict" type="boolean" default="true">
          Whether to enforce the schema strictly.
        </ParamField>
      </Expandable>
    </ParamField>
  </Expandable>
</ParamField>
