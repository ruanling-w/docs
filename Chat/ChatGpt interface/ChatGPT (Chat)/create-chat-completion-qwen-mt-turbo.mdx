---
title: "Create Qwen MT Turbo Chat"
api: "POST /v1/chat/completions"
description: "Generates a chat completion with machine translation capabilities using Qwen MT Turbo."
---

Qwen MT Turbo provides specialized translation services integrated into the chat completion workflow.

### Request Body

<ParamField body="model" type="string" required>
  Set to `qwen-mt-turbo`.
</ParamField>

<ParamField body="messages" type="array" required>
  Conversation history.
</ParamField>

<ParamField body="translation_options" type="object">
  Options for machine translation.
  <Expandable title="properties">
    <ParamField body="source_lang" type="string" default="auto">
      The source language of the text.
    </ParamField>
    <ParamField body="target_lang" type="string" required>
      The target language for translation.
    </ParamField>
  </Expandable>
</ParamField>
