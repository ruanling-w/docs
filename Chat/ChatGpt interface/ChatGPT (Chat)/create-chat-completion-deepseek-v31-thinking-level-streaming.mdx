---
title: "Create DeepSeek V3.1 Thinking Chat (Streaming)"
api: "POST /v1/chat/completions"
description: "Generates a streaming chat completion with deep reasoning using DeepSeek V3.1."
---

DeepSeek V3.1 supports advanced reasoning capabilities through the `thinking` field.

### Request Body

<ParamField body="model" type="string" required>
  The model ID (e.g., `deepseek-v3-1-250821`).
</ParamField>

<ParamField body="messages" type="array" required>
  Conversation history.
</ParamField>

<ParamField body="thinking" type="object">
  Configuration for deep reasoning.
  <Expandable title="properties">
    <ParamField body="type" type="string">
      `enabled`, `disabled`, or `auto`. Forces deep reasoning state.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="stream" type="boolean" default="true" required>
  Set to `true` to enable streaming.
</ParamField>
