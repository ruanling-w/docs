---
title: "Create Chat Function Call"
api: "POST /v1/chat/completions"
description: "Generates a chat completion that includes tool/function calls."
---

Function calling allows models to connect to external tools and APIs by generating structured JSON arguments.

### Request Body

<ParamField body="model" type="string" required>
  The model ID (e.g., `gpt-4o`).
</ParamField>

<ParamField body="messages" type="array" required>
  Conversation history.
</ParamField>

<ParamField body="tools" type="array" required>
  A list of tools the model may call.
  <Expandable title="tool object">
    <ParamField body="type" type="string" required>
      Set to `function`.
    </ParamField>
    <ParamField body="function" type="object" required>
      <Expandable title="function properties">
        <ParamField body="name" type="string" required>
          The name of the function.
        </ParamField>
        <ParamField body="description" type="string">
          A description of what the function does.
        </ParamField>
        <ParamField body="parameters" type="object">
          The parameters the function accepts, defined as a JSON Schema.
        </ParamField>
      </Expandable>
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="tool_choice" type="string" default="auto">
  Controls which function is called (`none`, `auto`, `required`).
</ParamField>
