# Image Object

Represents the URL or content of an image generated by the OpenAI API.

| Parameters | Type | Description |
| ------ | ------ | ------|
| b64\_json | string | If response\_format is b64\_json, the generated image is a base64 encoded JSON. |
| url | string | If response\_format is url (default), the generated image's URL. |
| revised\_prompt | string | If there are any revisions, the prompt used to generate the image. |

```JSON
{
  "url": "...",
  "revised_prompt": "..."
}
```
